# Google連携のセットアップ（約5分・無料）

このアプリの予定（授業・単発予定・課題の期限）を **Googleカレンダーに自動反映**し、
**iPhoneのショートカットから勉強タイマーの結果を送れる**ようにします。
中継には自分のGoogleアカウント上の Apps Script を使うので、外部サービスは不要です。

## 手順1：Apps Script を作る

1. ブラウザで **https://script.google.com** を開く（自分のGoogleアカウントでログイン）
2. 「＋ 新しいプロジェクト」を押す
3. エディタに最初から入っているコードを全部消し、
   このフォルダの **`gas/gas_bridge.gs`** の中身を丸ごと貼り付ける
4. 上のほうにある `const TOKEN = 'CHANGE_ME';` の **CHANGE_ME を自分だけの合言葉**に変える
   （例：`const TOKEN = 'kureha-9741-himitsu';`）
5. 💾（プロジェクトを保存）→ プロジェクト名は「FocusCafe Bridge」など適当でOK

## 手順2：ウェブアプリとして公開する

1. 右上の「**デプロイ**」→「**新しいデプロイ**」
2. 歯車マーク →「**ウェブアプリ**」を選ぶ
3. 設定：
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**（合言葉で守るのでOK）
4. 「デプロイ」→ 初回は「アクセスを承認」が出るので自分のアカウントで許可
   （「このアプリは確認されていません」と出たら「詳細」→「（プロジェクト名）に移動」）
5. 表示された **ウェブアプリのURL（https://script.google.com/macros/s/...../exec）をコピー**

## 手順3：アプリに設定する

1. アプリの **ライフタブ → 🗓カレンダー → 「☁️ Google連携の設定」** を開く
2. 手順2のURLと、手順1で決めた合言葉（トークン）を貼り付けて「💾保存」
3. 「📤 今すぐ同期」を押す → Googleカレンダーに **「FocusCafe」カレンダー**ができて予定が入る
4. スマホのGoogleカレンダーアプリで表示されない場合：
   メニュー → 設定 → FocusCafe を表示ON（初回のみ）

以後は6時間ごとに自動同期されます（「自動同期」チェックON時）。
予定・課題・休講・曜日振替を変えたら「📤今すぐ同期」で即反映できます。

## 手順4：iPhoneショートカット（勉強タイマー）

**「勉強スタート」ショートカット：**
1. ショートカットアプリ → ＋ → 名前「勉強スタート」
2. アクション「**現在の日付**」を追加
3. アクション「**ファイルを保存**」を追加（検索: ファイルを保存）
   - 保存先：iCloud Drive/Shortcuts、パス：`focus_start.txt`、「上書き」をON

**「勉強ストップ」ショートカット：**
1. ＋ → 名前「勉強ストップ」
2. 「**ファイルを取得**」（iCloud Drive/Shortcuts の `focus_start.txt`）
3. 「**日付を計算**」→「現在の日付」と「ファイル（の日付）」の**間の時間**を「**分**」で取得
4. 「**入力を要求**」→ 質問「何をやった？」（テキスト・デフォルト「勉強」）
5. 「**URLの内容を取得**」：
   - URL：手順2のウェブアプリURL
   - 方法：**POST**／本文：**JSON**
   - フィールド：`token`=合言葉、`action`=`log`、`task`=（手順4の回答）、`minutes`=（手順3の分）
6. （任意）「通知を表示」→「記録しました」

ホーム画面やロック画面ウィジェット・背面タップに登録すると1タップで使えます。
記録は Googleドライブの `FocusCafeSync/phone_inbox.json` に届き、
デスクトップのアプリが**5分おきに自動で取り込みます**（PCが起動しているとき。
起動していなくても、次にPCを付けたときにまとめて取り込まれます）。

## 手順5：iPhoneから「支出」「予定」を入力する

※ この機能は gas_bridge.gs の更新が必要です。すでにデプロイ済みの場合は、
最新の gas_bridge.gs を貼り直して「デプロイ」→「デプロイを管理」→ ✏️ →
バージョン「新バージョン」→「デプロイ」で反映してください（URLは変わりません）。

**「💸 支出を記録」ショートカット：**
1. 「**入力を要求**」→ 質問「いくら使った？」・入力の種類「**数字**」
2. 「**リストから選択**」→ リストに自分のカテゴリを登録
   （食費／外食／交通／日用品／勉強・自己投資／固定費／趣味・娯楽／衣服／健康・医療／その他）
3. 「**入力を要求**」→ 質問「何に使った？」・テキスト（空のままでもOK）
4. 「**URLの内容を取得**」→ ウェブアプリURL・方法 **POST**・本文 **JSON**：
   - `token`（テキスト）= 合言葉
   - `action`（テキスト）= `add`
   - `type`（テキスト）= `money`
   - `amount`（**数字**）= 手順1の回答
   - `category`（テキスト）= 手順2で選択した項目
   - `detail`（テキスト）= 手順3の回答

**「📌 予定を追加」ショートカット（複数日対応）：**
1. 「**入力を要求**」→ 質問「予定の名前は？」・テキスト
2. 「**日付を要求**」→ 質問「開始日は？」・種類「日付」
3. 「**日付をフォーマット**」→ カスタム `yyyy-MM-dd`（手順2の日付を変換）
4. 「**日付を要求**」→ 質問「終了日は？（1日だけなら同じ日）」・種類「日付」
5. 「**日付をフォーマット**」→ カスタム `yyyy-MM-dd`（手順4の日付を変換）
6. 「**入力を要求**」→ 質問「開始時刻（例 19:00）」・テキスト
7. 「**入力を要求**」→ 質問「終了時刻（例 20:30）」・テキスト
8. 「**URLの内容を取得**」→ POST・JSON：
   - `token` = 合言葉／`action` = `add`／`type` = `event`
   - `title` = 手順1／`date` = 手順3／`endDate` = 手順5
   - `start` = 手順6／`end` = 手順7

複数日の予定（旅行・インターン等）は、開始日〜終了日の**毎日に同じ予定**として
登録される（タイムライン・カレンダー・Googleカレンダーすべてに全日ぶん出る）。

**「📚 課題メモ」ショートカット（課題の確認・メモ・進捗をスマホから）：**

スマホで課題の**状態を見る**のは、Googleカレンダーの「📋 課題ボード」
（今日の終日イベント。全未完了課題の進捗・期限・メモが説明欄に入っている）を開くだけ。
**書き込む**のはこのショートカットで：

1. 「**入力を要求**」→ 質問「どの課題？（名前の一部でOK）」・テキスト
2. 「**入力を要求**」→ 質問「メモ（空OK）」・テキスト
3. 「**入力を要求**」→ 質問「進捗%（空欄=変えない）」・テキスト
4. 「**URLの内容を取得**」→ POST・JSON：
   - `token` = 合言葉／`action` = `add`／`type` = `asg_note`
   - `title` = 手順1／`note` = 手順2／`progress` = 手順3

名前が一致する課題があれば **メモ追記（📱時刻つき）＋進捗更新**、
一致しなければ**新しい課題として作成**される（期限は7日後・あとでPCで直せる）。
期限2日以内なら「今日絶対やる」にも自動反映。

**カレンダーに何を送るかの選択**：アプリのライフ→カレンダー→「☁️ Google連携の設定」で、
授業（1日1件の「○曜日課」まとめ／コマごと／送らない）・📌予定・📚課題期限・📋課題ボード・
💼就活のON/OFFを選べる。時間割に**教室**を入れておくと、日課まとめ・コマごとの説明欄に表示される。

どちらも Googleドライブの `FocusCafeSync/phone_inbox.json` に届き、
PC側が**5分おきに自動で取り込みます**。支出はアプリで手入力したのと同じ形
（判定つき）でマネータブへ、予定はライフのタイムライン・カレンダー・
Googleカレンダー同期にそのまま反映されます。PCが起動していない間の入力も、
次にPCを付けたときにまとめて取り込まれます。

## PC間のデータ共有（デスクトップ⇄ノート）

- 使用中は**5分おきに自動で記録がGドライブへ保存**されます（手動バックアップ不要）。
- もう一方のPCで `start_all.bat` を実行すると、Gドライブ側が新しい場合に
  「取り込みますか？ [y/N]」と聞かれるので **y** → 前のPCで入力した内容が引き継がれます。
- **2台で同時に使わないでください**（後から保存された方の内容が残ります）。

## 仕組みメモ

- カレンダーへは説明欄に `fcid:` タグを付けて書き込み、同期のたびに
  タグ付きイベントだけを入れ替えます。**手動で入れた予定は消えません**。
- 合言葉（トークン）が一致しないリクエストは Apps Script 側で拒否されます。
